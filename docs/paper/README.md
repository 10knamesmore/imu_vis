## 撰写约定

- **语言**：中文学术风格，被动或中立表述为主，避免第一人称
- **数据精度**：位置 cm（小数点 2 位），时间 s（1 位），角度度（1 位）
- **指标命名**：
  - `Loop closure error` / 闭环误差
  - `Drift rate` / 漂移速率
  - `Nearest-Point Distance (NPD)` / 最近点距离
  - `Mean NPD` / 平均 NPD
  - `RMSE NPD` / NPD 均方根
  - `Max NPD` / NPD 最大值
- **参考文献**：所有外部引用保留在各章节末尾的"参考文献"子节
- **图表引用**：使用 `../figures/<name>.png` 相对路径，转 Word 时只需在插图时替换
